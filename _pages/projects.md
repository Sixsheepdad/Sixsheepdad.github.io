## EasyTrain-Platform

An integrated platform for remote sensing image segmentation — from dataset management to model training and prediction.

**Key Features:**
- 🗂️ Dataset upload, preprocessing, and train/val/test splitting
- 🧠 Model registration with automatic metadata extraction
- 🏋️ Flexible training runs with configurable optimizers, schedulers, and augmentations
- 🔮 Inference & evaluation with mask output (index & RGB)
- ⏱️ Async job support — submit tasks without blocking the UI

**Tech Stack:** Python · FastAPI · PyTorch · uvicorn

🔗 [GitHub](https://github.com/Sixsheepdad/EasyTrain-Platform)

## Bilibili GIF Converter 🎬

A desktop app that downloads Bilibili videos and converts them to GIFs with ease.

**Key Features:**
- 📥 Download any Bilibili video via URL
- ✂️ Time & area cropping — select exactly the segment you want
- 🎨 Quality presets (low/medium/high) or custom FPS
- 🖼️ High-quality GIF output with FFmpeg palette optimization

**Tech Stack:** Electron · React 18 · yt-dlp · FFmpeg

🔗 [GitHub](https://github.com/Sixsheepdad/bilibili-gif-converter)
